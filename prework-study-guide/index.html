HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The h1 through h6 represents headers in sub catagory.
  • Make sure to save everytime and not lose work.

CSS

  • This is cascading style sheets.
  • Used to style HTML documents.
  • It describes how web pages should look with color, font, etc.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • Git is accessed by GitBash or GitHub website
  • Instsntly share code, notes, snippets, and projects.
  • Can work in cloud or locally.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • JavaScript is a lightweight, cross-platform, single-threaded, and interpreted compiled programming language.
  • It is also known as the scripting language for webpages.
